🗺️ Land Use & Classification Map

📘 What is a Land Use & Classification Map?

This map distinguishes between Forestlands, Alienable and Disposable Lands, and Protected Areas. It is essential for land planning, development permits, academic research, and legal documentation.

📋 Requirements

  • Valid Reason or Project Description
  • Target Municipality or Site
  • Preferred Map Format (e.g., PDF, Shapefile)
  • Endorsement Letter (if applicable)
